# Sudoku Game

A simple, browser-based Sudoku game built with HTML, CSS, and JavaScript.

## Features

- **Three Difficulty Levels**: Easy (35 clues), Medium (27 clues), Hard (22 clues)
- **Timer**: Track your solving time with start/pause functionality
- **Hidden Board**: Puzzle is blurred until you click Start for fair play
- **Input Methods**: Use keyboard (1-9) or click the number pad
- **Instant Feedback**: Green flash for correct, red for wrong entries
- **Celebrations**: Light blue animation when you complete a row, column, or 3x3 box
- **Reset & New Game**: Start over or try a different puzzle

## How to Play

1. Open `index.html` in any web browser
2. Select a difficulty level (Easy, Medium, or Hard)
3. Click **Start** to reveal the puzzle and begin the timer
4. Click an empty cell to select it
5. Enter a number (1-9) using your keyboard or the number pad
6. Fill all cells correctly to win!

## Controls

- **Click**: Select a cell
- **1-9 Keys**: Enter a number
- **Delete/Backspace**: Clear a cell
- **Arrow Keys**: Move between cells

## Files

- `index.html` - The complete game (single file, no dependencies)
